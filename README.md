# Tic Tac Toe

The game have 2 modes: 1v1 & 1vsComputer.

<p align="right">
  <img src="./assets/screenshot.png" width="80">
</p>
In order to play vs Computer, you should see a `💡` next to `Computer` (example: `Computer 💡`)

> [!IMPORTANT]
> The score is reset on page refresh.
> It does not reset automatically when changing the opponent type (player/computer)
