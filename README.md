# Tic Tac Toe


<img src="https://cdn-icons-png.flaticon.com/512/3405/3405019.png" align="right" width="120" height="178">

The game have 2 modes: `1v1` and `vs Computer`  

In order to play vs Computer, you should see a `💡` next to `Computer` (example: `Computer 💡`)  


> [!IMPORTANT]
> The score resets when the page is refreshed.  
> It does not automatically reset when changing the opponent type.

Play it [**here**](https://loghindev.github.io/tic-tac-toe/) 🚀
